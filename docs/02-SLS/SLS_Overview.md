## The Model
Clients export CSV from their ERP → SLS processes server-side → client gets a structured diagnostic report. The engine never ships. This is called **"holding the calculator."**
- Protects IP
- Justifies recurring monthly fees
- Limits liability
- No client-side software or IT involvement required

Pricing: one-time setup fee + monthly service fee. Four tiers (Starter → Enterprise). Not published on website — quote-based. Hourly oversight at $175/hr.
